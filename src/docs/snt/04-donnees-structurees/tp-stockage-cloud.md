# Stockage sur le cloud

## Introduction

Ces travaux pratiques ont pour objectif l'étude du fonctionnement d'un service de stockage de fichiers sur le Cloud.
Nous utiliserons le service **Mon drive Monlycée** de l'ENT pour lequel tout élève de la région Île-de-France dispose d'un accès et d'un espace de stockage de **100 Go**.

## Préparation

### Ordinateurs

Vous allez travailler à la fois sur votre ordinateur portable et l'ordinateur fixe du lycée.

!!! note "Instructions"

    1. Connectez-vous à l'ordinateur du lycée
    2. Revenez sur votre ordinateur portable et effectuez les travaux pratiques jusqu'à ce qu'il vous soit demandé d'utiliser l'ordinateur du lycée

### Activation sur l'ordinateur

!!! target "Objectif"
    
    S'assurer du bon fonctionnement de la synchronisation sur votre ordinateur portable.
    Si n'avez pas votre ordinateur portable et que vous n'utilisez que l'ordinateur du lycée, passez directement à la section [:material-link: Activation sur le téléphone](#activation-sur-le-telephone)

!!! note "Instructions"

    1. Lancez l'application **Drive monlycée.net**
    2. Cliquez sur **Suivant**, le navigateur web devrait s'ouvrir en vous demandant de vous connecter à votre compte ENT
    3. Suivez les instructions afin d'autoriser l'accès à votre compte ENT
    4. La synchronisation devrait se lancer automatiquement et un dossier **Drive monlycee.net** devrait être visible dans l'explorateur de fichiers de votre ordinateur portable

??? warning "Vous n'avez pas l'application **Drive monlycée.net**"

    Il est possible de télécharger cette application depuis l'ENT :

    1. [:material-link: Connectez-vous à l'ENT](https://ent.iledefrance.fr/){:target="_blank"}
    2. Accédez à l'application **Mon drive Monlycée**
    3. Rendez-vous dans le dossier **Dossiers de groupes**
    4. Rendez-vous ensuite dans **Guide utilisateur** puis **Application Windows**
    5. Téléchargez et installez l'application

### Activation sur le téléphone

!!! target "Objectif"
    
    Installer l'application **Drive monlycée.net** sur le téléphone afin de pourvoir l'utiliser pour consulter ou déposer des documents.

!!! note "Instructions"

    1. Prenez votre téléphone portable
    2. Rechechez et installez l'application **Drive monlycée.net** depuis le magasin d'applications de votre téléphone
    3. Lancez l'application **Drive monlycée.net**
    4. Connectez-vous à l'ENT à la demande de l'application


## Mise en pratique

### Accès depuis le Web

!!! target "Objectif"
    
    Accéder à vos documents stockés sur le *cloud* depuis n'importe quel appareil.

Imaginons que vous utilisiez un ordinateur sur lequel il vous est impossible d'installer le service **Drive monlycée.net**.
Pour contourner ce problème, les services de stockage sur le *cloud* fournissent un accès depuis une interface web.

!!! note "Instructions"

    1. Utilisez maintenant l'ordinateur du lycée
    2. Lancez un navigateur web et [:material-link: Connectez-vous à l'ENT](https://ent.iledefrance.fr/){:target="_blank"}
    3. Accédez à l'application **Mon drive Monlycée**
    4. Déposez le contenu de votre dossier personnel du réseau du lycée dans le dossier sur **Mon drive Monlycée**
    5. Vérifiez que les fichiers déposés sont bien visibles depuis votre ordinateur portable et votre téléphone

### Partage

!!! target "Objectif"
    
    Partager un fichier ou un dossier à un autre utilisateur.

Les services de stockage *cloud* disposent généralement de fonctions de partage des dossiers et des fichiers.
Vous pouvez également autoriser des utilisateurs à modifier les dossiers et les fichiers partagés.

!!! note "Instructions"

    1. Accédez à l'application **Mon drive Monlycée** depuis un navigateur web
    2. Créez un dossier intitulé `Test partage`
    3. Cliquez sur l'icône de partage du dossier (l'icône en forme de personne avec un +)
    4. Partagez le dossier à l'un de vos camarades en saisissant son adresse e-mail ENT


### Test des fonctionnalités

Maintenant que vous vous êtes familiarisé avec un service de stockage sur le *cloud*, effectuez l'ensemble des actions suivantes :

#### En utilisant l'ordinateur
- Créer un nouveau document texte et le modifier depuis l'interface web (ou modifier un document existant)
- Créer un nouveau document feuille de calcul et le modifier depuis l'interface web
- Créer un nouveau document présentation et le modifier depuis l'interface web 

#### En utilisant l'application mobile
- Téléverser une photo ou une vidéo
- Scanner un document papier
- Créer un mémo vocal
